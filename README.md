Description

This project is a command-line based calculator application developed using Python.
It performs basic arithmetic operations such as addition, subtraction, multiplication, and division based on user input.

This task is completed as part of the Python Programming Internship by MyDailyWork.
Features

Takes two numeric inputs from the user

Supports basic arithmetic operations:

Addition (+)

Subtraction (−)

Multiplication (×)

Division (÷)

Handles division by zero safely

Simple and user-friendly CLI interface
